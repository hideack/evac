
### pushbullet

Out to [Pushbullet](https://www.pushbullet.com/).

#### Parameters

| Name   | parameter |
|--------|----------------------------|
| device  | Target device. |
| token   | Pushbullet access token. https://www.pushbullet.com/account |
| title   | Push title. |
| mode    | "note" or "link" |

#### Example

    "pushbullet": {
      "device": "*****@gmail.com",
      "token" : "*****",
      "title": "test",
      "mode": "link"
    }
