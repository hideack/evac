evac
================

[![Build Status](https://travis-ci.org/hideack/evac.svg?branch=master)](https://travis-ci.org/hideack/evac)

node.js based simple aggregator.

### Plugin
#### input
- mysql
- rss
- static word
- web page parser

#### filter
- diff
- match
- through

#### output
- http post
- ikachan
- mail
- stdout
- shell
